📍 StudySpot (스터디스팟)
"나의 학습 스타일과 목적에 딱 맞는 최적의 공부 장소 추천 플랫폼"

📝 프로젝트 개요
StudySpot은 사용자의 학습 목적(암기, 과제, 코딩 등)과 선호도에 맞춰 최적의 장소(카페, 도서관, 스터디룸 등)를 추천하고, 이용 경험을 공유할 수 있는 웹 서비스입니다. 단순 정보 제공을 넘어 맞춤형 검색-상세 조회-태그 기반 리뷰-개인화 관리로 이어지는 서비스의 유기적인 흐름을 안정적으로 구현하는 데 집중했습니다.

👥 팀 구성 및 역할
총 인원: 2명 (풀스택 협업)

역할 분담:

이지수 (PM & PL): 프로젝트 기획 및 전체 리딩, 전체 DB 설계(ERD/SQL) 및 아키텍처링, 검색/추천 엔진 로직 설계 및 백엔드 메인 개발

문이화: 리뷰 시스템 구현, 마이페이지 관리 기능 및 테스트 진행

✨ 주요 기능 (MVP)

상세 정보 조회: 장소의 상세 설명, 시설 정보 및 공부 적합도 시각화 정보 제공

태그 기반 리뷰: 사용자의 실제 경험을 바탕으로 한 태그 입력 및 후기 작성 기능

마이페이지: 본인이 작성한 리뷰 목록 조회 및 개인 선호도 설정 관리

회원 인증: 일반 로그인 및 사용자 세션 관리

🛠️ 기술 스택
Backend: Java / Spring Boot

Frontend: HTML5 / CSS3 / JavaScript

Database: MySQL (JPA/Hibernate)

Tools: Git / GitHub, SourceTree

🎯 기술적 접근
1. 데이터 정합성을 고려한 DB 설계 
서비스의 확장성과 데이터 무결성을 위해 전체 도메인 간의 관계를 직접 분석하고 ERD를 설계했습니다.

특히 장소 속성 정보와 사용자 리뷰 태그 간의 연관 관계를 정교하게 정의하여, 검색 시 정확도 높은 결과를 반환할 수 있도록 SQL 쿼리를 최적화했습니다.

2. 태그 기반 데이터 구조화
사용자 리뷰를 정형화된 태그 데이터로 수집하여, 향후 추천 로직의 가중치로 활용할 수 있도록 데이터 구조를 설계했습니다. 이는 단순 텍스트 검색보다 사용자 맞춤형 추천의 신뢰도를 높이는 기반이 됩니다.

3. 협업 및 코드 품질 관리
GitHub와 SourceTree를 활용해 브랜치 전략을 수립하고, 정기적인 코드 리뷰를 통해 시스템의 전체적인 데이터 흐름을 함께 파악하며 개발 효율을 극대화했습니다.

🚀 향후 고도화 계획
현재의 장소 추천 및 리뷰 기능을 기반으로 아래의 기능을 추가할 예정입니다.

실시간 예약 시스템: 날짜 및 시간 슬롯 선택을 통한 스터디룸 예약 기능

예약 상태 자동화: 스케줄러를 활용한 이용 상태(예정/완료) 자동 변경 로직

위치 기반 시각화: 지도 API 연동을 통한 내 주변 장소 거리순 조회
